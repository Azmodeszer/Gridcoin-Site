Earn Gridcoin for Folding@Home

  1. Follow our solo crunching guide and attach to one or more BOINC projects. You do not need to actually crunch any work, you just need BOINC installed and attached to a project. In BOINC Manager in advanced view, you can set the project to "no new tasks".
  2. Find your CPID, this is a unique identifier assigned to you by BOINC projects. You can find it in the Gridcoin wallet under hamburger menu > settings > researcher wizard > projects.
  3. Download the Folding@home client from Folding@Home and start installation
  4. When prompted, create a username with the format: friendlyname_GRC_CPID. Friendlyname can be any name you want, GRC must remain in the middle and be capitalized, CPID should be your CPID
  5. Not required but helps promote Gridcoin: enter team # 226804 when prompted
  6. Setting up a passkey is optional but strongly suggested as you will get more F@H credit which means more GRC
