Steps to Earn GRC via DPOR

  • 1) Check Your Projects
    • a) Make sure any projects you crunch are on the whitelist. If it isn't on it, you will not be rewarded for it
    • b) Check that each project crunched is on team Gridcoin. You will not get rewarded if they are not on the team
    • c) Check that the CPIDs (cross project identifier) are the same across your projects. You can check on each project or check using boincstats
      • If you found differing CPIDs, use this guide
    • d) if you just started crunching, you may need to wait around 24 hours for statistics to sync
  • 2) Configure the Wallet
    • a) In the wallet, go to the overiew page and check the message in the lower left. Makes sure it doesn't say INVESTOR
      • If you see INVESTOR, check that you did all steps in Check Your Projects. If you setup BOINC in a non-default path, add it in the config (see Step 2)
      • Note: BOINC is needed on the machine running the wallet. It needs to be connected to any project on your account (it can be suspended)
    • b) Send a beacon
      • I) Unlock your wallet fully if you've locked it
      • II) Go to the debug tab, then console, and enter "advertisebeacon" without quotes. A transaction should be sent
      • III) After it's been sent, Go to the console and run "beaconreport" without quotes. Check for your CPID. You may need to wait ~24 hours
      • IV) You can now lock your wallet again
  • 3) Running the Wallet to Stake
    • a) Unlock the wallet for staking only if locked (is recommended to have it locked for security)
    • b) Run the wallet close to 24/7. When your wallet is off, you miss chances to stake blocks and earn rewards
    • c) Every 6 months, you will need to send a beacon again. Repeat the process shown in Configure the Wallet step b
    • d) Keep the wallet up to date as updates are released
